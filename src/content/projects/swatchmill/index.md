## SwatchMill

SwatchMill is an Adobe Illustrator plugin built for fashion designers and production teams. It automates the most tedious part of the design-to-production handoff: pulling a color palette from artwork, matching it against a swatch library, and generating mill sheets — all in a single workflow.

The plugin analyzes selected artwork to identify and extract colors (from both vector graphics and embedded Photoshop files), matches them against imported Adobe Swatch Exchange (.ASE) libraries, and outputs production-ready mill sheets, repeat fills, and swatch artboards.

## Features

- Color extraction from artwork — vector and embedded Photoshop files
- ASE swatch library import and color matching with coverage percentages
- One-click generation of mill sheets, repeat fills, and swatch artboards
- Eyedropper tool for manual color adjustment
- Minimum coverage threshold filter to clean up results
- One-time purchase, no subscription — distributed via Adobe Exchange

## Built for

Fashion brands and production designers who previously managed color matching and mill sheet creation by hand. SwatchMill cuts a multi-step manual process down to a few clicks, letting designers stay focused on the work rather than the paperwork.
